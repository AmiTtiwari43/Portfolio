# 🚀 Portfolio Website 2026

A modern, high-performance, and visually stunning 3D portfolio website built with **Next.js 15**, **React 19**, and **Three.js**. This project showcases advanced web development techniques, including real-time 3D rendering, fluid animations, and a responsive bento-grid layout.

---

## ✨ Features

- **🌐 Interactive 3D Environment**: Immersive 3D scenes powered by Three.js and React Three Fiber.
- **🎨 Modern UI/UX**: Futuristic design featuring glassmorphism, smooth gradients, and a Bento-style skills grid.
- **⚡ High Performance**: Built on Next.js 15 with App Router for optimized loading and SEO.
- **🎬 Fluid Animations**: Dynamic transitions and micro-interactions using GSAP and Framer Motion.
- **📱 Fully Responsive**: Seamless experience across all devices, from mobile to desktop.
- **🛰️ Smooth Scrolling**: Integrated Lenis scroll for a premium feel.

---

## 🛠️ Tech Stack

### Frontend & Core
- **Next.js 15**: React framework for server-side rendering and static site generation.
- **React 19**: The latest version of the core UI library.
- **TypeScript**: Ensuring type safety and robust code architecture.

### 3D & Graphics
- **Three.js**: The foundation for 3D graphics on the web.
- **@react-three/fiber**: React renderer for Three.js.
- **@react-three/drei**: Useful helpers and abstractions for R3F.
- **@react-three/postprocessing**: Advanced visual effects and filters.

### Styling & UI
- **Tailwind CSS 4**: Modern utility-first CSS framework for rapid styling.
- **Lucide React**: Beautifully simple pixel-perfect icons.
- **Devicon**: Visual representations of technical skills and tools.

### Animation & Motion
- **GSAP (GreenSock Animation Platform)**: Professional-grade animations.
- **Framer Motion**: Declarative animations for React components.
- **Lenis**: High-performance smooth scrolling.

## 📂 Project Structure

```text
├── app/              # Next.js App Router (pages and layouts)
├── components/       # Reusable UI, Layout, and 3D components
│   ├── 3d/           # Three.js specific components
│   ├── layout/       # Main page sections (Hero, About, Projects, etc.)
│   └── ui/           # Small, reusable UI elements (Buttons, Navs)
├── scenes/           # 3D Scene definitions
├── animations/       # Animation logic and constants
├── hooks/            # Custom React hooks
├── lib/              # Utility functions and shared logic
├── public/           # Static assets (images, models, icons)
└── stores/           # State management stores
```

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AmiTtiwari43/Portfolio.git
   cd Portfolio-Website-2026
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
